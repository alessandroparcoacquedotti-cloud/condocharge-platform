# Changelog

All notable changes to Condo Charge are documented in this file.

## v1.2.0 Planned Analytics

Planned evolution focused on visibility, reporting, and decision support for condominium administrators.

### Planned

- Advanced analytics for charging usage trends
- Richer reporting views for condominium operations
- Historical insight tools for service optimization
- More product-facing KPI storytelling for administrators

## v1.1.0 Planned Smart Queue

Planned product milestone focused on smarter access orchestration for shared charging infrastructure.

### Planned

- Smart Queue for shared station demand
- Prenotazione intelligente
- Timer di disponibilita
- Clearer user flows for shared charging coordination

## v1.0.0 Production Pilot

Initial production pilot release focused on condominium EV charging operations and product presentation.

### Added

- Multi-tenant condominium architecture with tenant-aware data isolation
- Resident portal for station visibility, charging history, and service transparency
- Admin portal for operational oversight, resident management, and RFID assignment
- Billing periods, statement generation, and reconciliation workflow
- Push and Telegram communication flows for service notifications
- Health monitoring visibility for agent heartbeat and polling continuity
- Legrand integration for station status and local agent synchronization

### Release Notes Summary

- Condo Charge is positioned as a professional product repository, not only as a codebase
- The repository includes CI, screenshots, polished release documentation, and demo-oriented materials
- The production pilot story covers resident experience, admin control, notifications, health visibility, and condominium service management
