# Changelog

All notable changes to CondoCharge are documented in this file.

## v1.0.0

Initial public portfolio release focused on condominium EV charging operations.

### Added

- Multi-tenant condominium architecture with tenant-aware data isolation
- Resident portal for charging visibility, session history, and billing review
- Admin portal for resident management, RFID assignment, and charging oversight
- Billing periods and resident statement generation
- Reconciliation workflow with payment import and unmatched-payment tracking
- PDF statement export for resident billing communication
- Email notification flows for reminders, receipts, and statements
- Legrand charging integration for station and session import workflows

### Release Notes Summary

- CondoCharge is ready for recruiter-facing and portfolio-facing presentation
- The repository includes CI, typed backend code, test coverage, screenshots, and deployment-oriented documentation
- The product demo story covers resident attribution, monthly billing, payment reconciliation, and resident self-service visibility
