Recommended tag: `v1.0.0`

## Release Summary

CondoCharge `v1.0.0` is the first public portfolio-ready release of the project.

It presents CondoCharge as a credible condominium EV charging platform with:

- multi-tenant architecture
- a resident portal
- an admin portal
- billing periods and resident statements
- reconciliation and payment tracking
- PDF statement generation
- email notifications
- Legrand charging integration

## Highlights

- Clear operational workflow from charging-session attribution through resident billing
- Tenant-safe role-based access control for admins and residents
- Recruiter-ready repository presentation with screenshots, architecture docs, and release documentation
- CI-backed code quality with linting, typing, tests, and frontend build verification

## Recommended Release Positioning

Use this release as the public baseline for:

- portfolio presentation
- recruiter review
- technical interviews
- architecture walkthroughs
- product and operations demos

## Suggested Release Description

CondoCharge is a condominium EV charging management platform that helps building operators attribute shared charging sessions to residents, generate billing statements, reconcile payments, and provide residents with transparent consumption and billing visibility.
