# Condo Charge v1.0.0

## Release Title

Production Pilot

## Summary

Condo Charge v1.0.0 introduces the product as a smart EV charging platform for condominiums.

This release positions Condo Charge as more than a station-status viewer: it is a product built to help condominium administrators manage charging as a clear, monitorable, and communicable service.

## Highlights

- Real-time station visibility
- Resident experience designed for clarity and mobile usage
- Admin dashboard for operational control
- RFID management
- Push notifications
- Telegram integration
- Health monitoring for the local Windows agent and polling continuity
- Billing and service-management foundations for the production pilot

## Why This Release Matters

Shared charging in condominiums creates a service-management problem, not only a hardware problem.

Condo Charge addresses that gap by connecting infrastructure, resident communication, operational monitoring, and administrative control in one coherent platform.

## Product Positioning

Condo Charge is **not** just an app to see charger status.

Condo Charge is a **smart platform for condominium charging management**.

## Pilot Scope

- Resident PWA
- Admin PWA
- Railway frontend
- Railway backend
- SQLite pilot storage
- Windows agent for Legrand polling and heartbeat
- Telegram and Web Push communication channels
